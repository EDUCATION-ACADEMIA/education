What do you need to put your web pages on the www?

(A) a web server

(B) a web browser

(C) a connection to internet

(D) ** all of the above

EXPLANATIONS BELOW

Concept note-1: -A. A connection to the internet. A web browser. A web server. All of the above.

Concept note-2: -Browser: A software application used to locate and display Web pages. The two most popular browsers are Microsoft Internet Explorer and Netscape Navigator.

Concept note-3: -The building blocks of the Web are web pages which are formatted in HTML and connected by links called “hypertext” or hyperlinks and accessed by HTTP. These links are electronic connections that link related pieces of information so that users can access the desired information quickly.

Concept note-4: -A web page can be accessed by entering a URL address into a browser’s address bar. A web page may contain text, graphics, and hyperlinks to other web pages and files. A web page is often used to provide information to viewers, including pictures or videos to help illustrate important topics.

Concept note-5: -Header & menu. The header is the uppermost part of a website. Images. Immediately below the header is some form of image, series of images or sometimes a video. Website content. All sites contain content. Footer. Simply put, a footer is the bottom most part of any site. 23-Mar-2021
