Where does a computer add and compare its data?

(A) Floppy disk

(B) ** Hard disk

(C) Memory

(D) CPU

EXPLANATIONS BELOW

Concept note-1: -The correct answer is CPU chip. The Central Processing Unit is the main part of the computer. CPU chip or main processor is that part in the computer which performs or executes all the instructions or programs.

Concept note-2: -Therefore, computers add and compare its data in the CPU.

Concept note-3: -By comparing one set of data with another, the computer can locate, analyze, select, reorder and make decisions. After comparing, the computer can indicate whether the data were equal or which set was numerically greater or less than the other.

Concept note-4: -Insert a chart to illustrate and compare data. It is used to illustrate comparisons of related numbers. There are different types of charts such as pie charts, bar charts, line charts, area charts, histograms, flowcharts, bubble charts, etc. which are used to compare data.

Concept note-5: -Hard Disk Drives: a device that reads and writes data to the hard disk. Data is copied from the computerâ€™s main memory (random-access memory or RAM), and then written to the hard disk. The drive is the mechanical device that writes the data to the disk.
